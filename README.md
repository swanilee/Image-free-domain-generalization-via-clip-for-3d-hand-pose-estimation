## Image free domain generalization via clip for 3d hand pose estimation

### Prerequisties
The code needs the following libraries:
* Python 3.6.13
* Anaconda 
* Pytorch 1.7.1


## Data Preparation
We use the dataset of FreiHAND for training, and STB, RHD for testing.
